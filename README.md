# Tuwanx Landing Page - Laravel Application

This is Tuwanx custom fashion marketplace landing page.

## Requirements

- PHP >= 8.2
- Composer
- SQLite (already configured)

## Running the Application

Start the Laravel development server:

```bash
php artisan serve
```

The application will be available at `http://localhost:8000`

## Available Routes

- `/` - Home page
- `/about` - About Us page
- `/contact` - Contact page with form
- `/privacy-policy` - Privacy Policy
- `/terms-and-condtitions` - Terms and Conditions

## Features

- Responsive navigation with mobile menu
- Contact form with validation and email capability
- Smooth scrolling animations
- All original styling preserved using Tailwind CSS
- FAQ accordion functionality

## Setup Notes

- All assets are in `public/assets/`
- Contact form requires mail configuration in `.env`
- No database required for basic functionality (uses SQLite)
